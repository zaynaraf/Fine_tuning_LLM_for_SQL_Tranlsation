# Results

This folder contains the human-readable evaluation report, the recorded fine-tuned evaluation notebook, and raw SQL artifacts from the Spider dev workflow.

## Files

| File | Description |
| --- | --- |
| `evaluation_report.md` | Recruiter-friendly summary of the recorded metrics and interpretation. |
| `report_ft_v3_spider_eval.ipynb` | Archived notebook output from the fine-tuned v3 Spider evaluation run. |
| `raw_spider_outputs/dev_gold.sql` | Spider dev gold queries with database ids. |
| `raw_spider_outputs/dev_pred_base_model.sql` | Predictions generated by the base `Qwen/Qwen2.5-Coder-1.5B-Instruct` model in the original base-model evaluation notebook. |
| `raw_spider_outputs/dev_gold_head.sql` | First 20 Spider dev gold queries used for a dry-run check. |
| `raw_spider_outputs/dev_pred_head_base_model.sql` | First 20 base-model predictions used for the dry-run check. |

The fine-tuned v3 prediction file was generated inside the ignored `spider_official_eval/` workspace during the completed notebook run, so it is not committed here. The recorded fine-tuned metrics are preserved in the project README and in `results/report_ft_v3_spider_eval.ipynb`.
